# 行业不存在或信息不足处理流程

当用户输入的行业不存在、行业配置缺失、海外公开信息不足，或无法生成有效候选池时，按以下流程处理。

## 1. 行业配置不存在

如果用户只输入行业名称，且当前项目没有该行业配置：

1. 不直接跑候选池。
2. 先生成行业配置草案。
3. 默认补齐：
   - 行业中文名
   - 品牌名
   - 目标区域
   - 可能的信息源分级
   - 纳入主题
   - 排除主题
   - 海报断行术语
4. 回复用户：配置草案已生成，请补充市场、关注点、排除内容和最终用途。

## 2. 行业边界模糊

如果行业名称过宽或含义不清，例如“制造业”“科技行业”“消费品行业”：

先要求用户收窄到可搜索、可筛选的行业边界。

建议提问：

```text
这个行业范围比较宽。请补充一个更具体的子行业，例如：
- 制造业：化工 / 机械设备 / 半导体设备 / 包装材料
- 科技行业：AI软件 / 云计算 / 消费电子 / 企业SaaS
- 消费品：宠物 / 美妆 / 母婴 / 家居用品
```

## 3. 信息源不足

如果搜索后无法找到足够高质量来源：

1. 不强行凑满 20 条低质量候选。
2. 输出“可生成数量”和“缺口原因”。
3. 用最近 30-90 天的高质量背景内容补足，但必须标注“背景补充”。
4. 如果仍不足 12 条，停止生成候选池，请用户补充信息源或放宽条件。

回复格式：

```text
当前行业公开信息不足，暂不建议生成正式候选池。

已找到高质量候选：{n}条
主要缺口：
- 权威媒体少
- 新闻更新频率低
- 信息多为企业软文
- 与目标区域关联弱

建议补充：
- 2-5个指定媒体或协会
- 重点公司名单
- 目标国家/区域
- 是否允许使用企业公告或展会信息
```

## 4. 信息质量不达标

以下情况不能进入候选池：

- 只有自媒体转述，无原始来源。
- 只有企业宣传口径，没有行业影响。
- 日期不清楚。
- 地区不清楚。
- 与用户关注点弱相关。
- 搜索结果主要是股价、招聘、广告、黄页、展位销售页。

## 5. 降级生成规则

如果用户明确要求“先给我一个初版”，可降级生成：

- `10条观察线索`，而不是 `20条候选选题`
- 每条标注信息等级：
  - A：权威行业媒体/官方来源
  - B：企业公告/展会/协会
  - C：背景报告/市场摘要
- 明确说明：这不是正式候选池，不能直接进入最终快报。

## 6. 绝对禁止

- 禁止编造不存在的媒体、日期、公司动作或链接。
- 禁止为了凑数纳入弱相关内容。
- 禁止把泛财经新闻包装成行业热点。
- 禁止在用户未确认前进入最终海报或正式快报。
