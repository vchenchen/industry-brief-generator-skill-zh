# 候选池输出模板

标题：

```text
{brand}｜今日候选选题池｜{date}
```

## 今日选题主线

输出 3-5 条趋势判断：

- 趋势判断一。
- 趋势判断二。
- 趋势判断三。

## {candidate_count}个候选选题

每条格式固定：

```text
{n}. 【{n}｜{领域标签}｜{区域}】{标题}
   来源/日期：{source}，{date}
   一句话摘要：{summary}
   入选理由：{reason}
   适合角度：{angle}
   来源链接：{url}
```

## 排序规则

- 优先最近 24-48 小时。
- 再补最近 7 天仍有价值内容。
- 展会、政策、并购、价格、供需、渠道变化、新品代理机会优先。
- 软文、泛报告、弱相关内容只可补位。

## 结尾固定提醒

```text
请回复{final_count}个编号，我将据此生成今日快报海报、文字版快报和小红书文案。
```

若最终用途不是小红书，可替换为：

```text
请回复{final_count}个编号，我将据此生成今日快报海报和{final_use}版本。
```
